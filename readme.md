### for_redis

请看 ``for_redis``分支 来处理

>：这是一个引入redis储存聊天记录的分支。主分支为了保持简洁性，并未加入过多特性。但web应用的特性，会进行刷新导致历史记录的丢失。本分支便是为了解决这一问题。纯粹的使用服务器的内存容易造成程序过载导致崩溃。

请确保你的服务器开启了redis服务并且安装了PHP-redis扩展
